--> Project Title: EV Charging Station Finder
    Deployed Link: http://ev-plugfinder-env.eba-euajkitg.ap-south-1.elasticbeanstalk.com

--> Overview:
    The EV Charging Station Finder is a web-based and mobile application designed to assist electric vehicle (EV) owners in locating nearby charging stations efficiently. The platform aims to provide real-time information on the availability, type, and status of charging stations, helping users plan their routes and ensure their vehicles remain charged. The system is designed with a user-friendly interface and integrates advanced features to enhance the EV charging experience.

--> Objectives:

    -> Real-Time Information: Provide accurate, real-time data on the availability and status of charging stations.
    -> User-Friendly Interface: Develop an intuitive interface that makes it easy for users to find and navigate to charging stations.

--> Features:

    -> User Registration and Login:
        Secure user authentication and profile management.
        Option for social media login.

    -> Charging Station Search:
        Search filters based on location, charging type (e.g., fast, slow), connector type, and availability.
        List view with station details including address.

--> Technologies Used:

    -> Frontend:
       HTML, JavaScript, Tailwind CSS for responsive design.
       EJS Templeating

    -> Backend:
        Node.js and Express.js for server-side logic.
        MongoDB for database management.

    -> Deployment:
        Cloud service AWS for hosting.
